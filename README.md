# react-playground

Webpack + React.js + Babel 6 + Mocha + Enzyme

## 0. Up and Running 

Clone the repo: 

```
$ git clone git@github.com:applegrain/react-playground.git
```

Install the dependencies: 

```
$ npm install
```

Run the tests: 

```
$ npm test 
|| 
$ npm run test:watch
```

Start the dev server: 

```
$ npm run dev
```
